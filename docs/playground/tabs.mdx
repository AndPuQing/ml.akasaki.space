# Docusaurus Tabs 示例

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';


### 代码

````html
import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs groupId="language">
  <TabItem value="tf" default>

  ```py
  import tensorflow as tf
  # TensorFlow code
  # TensorFlow code
  # TensorFlow code
  # TensorFlow code
  # TensorFlow code
  ```

  </TabItem>
  <TabItem value="torch">

  ```py
  import torch
  # PyTorch code
  # PyTorch code
  # PyTorch code
  # PyTorch code
  ```

  </TabItem>
</Tabs>
````
